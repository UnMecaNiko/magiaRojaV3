# Salidas — contenido generado

Todo lo que se **genera a partir** de la fuente de la verdad (`conocimiento/` + `comercial/`). Nada aquí es fuente: si un dato de una salida contradice a `conocimiento/`, la fuente gana y la salida se regenera.

| Carpeta | Qué contiene |
|---|---|
| `articulos/` | Artículos y tutoriales del proceso (generados desde `proceso-construccion/` e `historia/`) |
| `web/` | Contenido y páginas web de la máquina |
| `chatbot/` | Corpus/contexto preparado para el chatbot de preguntas y respuestas |

## Reglas

1. Toda salida sigue la [guía de estilo](../harness/guias/estilo-redaccion.md).
2. Toda salida debe poder **regenerarse** desde la fuente — indicar al inicio de cada archivo de qué documentos fuente deriva.
3. Contenido para clientes: **sin precios**, claims trazables.
4. Las traducciones (futuro) viven aquí, nunca en la fuente (D-0001).
