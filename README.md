# CNC Magia Roja v3 — Base de Conocimiento

Máquina CNC de escritorio con láser de 30W ópticos y cabezal intercambiable para fresado/grabado. Tercera iteración de la línea **Magia Roja**, construida por el equipo padre-hijo **Saul Velasquez** (mecánica) y **Nicolas Velasquez** (electrónica y control).

Este repositorio es la **base de conocimiento central y única** de la máquina. De aquí se alimentan:

- 🤖 Modelos y agentes que generan **documentación y planos**
- 🌐 **Páginas web** y **artículos** que explican el proceso
- 💬 Un **chatbot** que responde preguntas sobre la máquina
- 📈 Material **comercial** (propuesta de valor, casos de uso — sin precios)
- 📚 **Tutoriales** paso a paso del proceso de construcción
